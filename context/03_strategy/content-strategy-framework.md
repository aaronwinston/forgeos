# Content Strategy Framework

## Purpose

Use this framework to align content across audiences, channels, and strategic themes.

It is not external messaging and should not be used alone to generate campaigns.

## North Star

Developer expertise matters more than ever in the age of AI.

## Core Message

Software development is changing quickly. Developers, teams, and organizations need clearer, faster, safer ways to work.

## Strategic Pillars

### Work Faster

Focus on velocity, flow, automation, and reducing friction.

### Work at Scale

Focus on collaboration, team workflows, platform leverage, and shared systems.

### Work More Securely

Focus on code quality, governance, remediation, trust, and safe defaults.

## Audience Model

- Builders — people turning ideas into working software
- Learners — people building skill and confidence
- Drivers — people enabling tools, policy, or platforms
- Leaders — people deciding investment, direction, or risk tolerance

## Objectives

- Awareness
- Education and acquisition
- Retention and re-engagement

## Agent Instruction

Use this as an alignment filter. Pair it with messaging and execution files before generating content.
