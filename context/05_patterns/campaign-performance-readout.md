# Campaign Performance Readout Pattern

## Purpose

Use this structure to turn a campaign into reusable organizational learning.

## Readout Structure

1. Overview
2. Strategic thesis
3. Core strategy
4. Creative system
5. Performance analysis
6. Key learnings
7. Strategic implications
8. Reusable principles
9. Final takeaway

## Useful Strategic Claims

- Developer marketing is developer enablement.
- Workflows outperform features.
- Interface is a strong marketing asset.
- Qualification can be more valuable than reach.
- Message continuity increases conversion.
- Specificity builds trust.

## Agent Instructions

When writing campaign readouts:

- Focus on learnings, not vanity metrics.
- Separate strategy from performance.
- Extract reusable principles.
- Make the next campaign better.
