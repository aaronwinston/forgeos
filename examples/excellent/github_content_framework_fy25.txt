# GitHub Content Marketing Framework FY25

## 🔑 How to Use This Document (Agent Context)

- This is a **strategic content framework**, not messaging or copy.
- Use it to **align outputs to narrative, audience, and pillars**.
- It defines the **why and direction**, not execution.
- Always combine with:
  - Messaging/positioning docs (what to say)
  - Execution frameworks (what to produce)

### Best uses
- Select a **pillar** (Speed, Scale, Security)
- Choose a **persona**
- Ensure **narrative consistency**

### Do NOT use for
- Writing copy directly
- Making product claims
- Acting as source of truth

### Rule
This document is a **constraint system, not a generator**.
