---

## 🎯 Core Positioning

Tagline: Agents for every app.

Positioning:
The Copilot SDK gives developers programmatic access to a production-tested agent execution engine, so they don’t have to build orchestration systems from scratch.

---

## 🧠 Core Insight

“Agents are everywhere. Production-grade agents are not.”

---

## 🧱 Messaging Pillars

- Proven runtime → don’t rebuild orchestration
- Multi-model → flexibility without complexity
- Decoupled execution → not tied to IDE
- MCP-native → extensible with real context

---

## ⚔️ Competitive Framing

- LLM APIs → generate text
- Other SDKs → primitives
- Copilot SDK → production-ready execution engine

- Embed agents anywhere
